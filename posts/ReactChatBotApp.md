JavaScript のフレームワークである React と firebase を用いて、**ChatBot アプリ**を作成しました。

## 使用技術

- React (v 17.0.2)
- FireBase (v 9.6.7)
- MaterialUI (v 5.4)

このアプリケーションは slack の API を用いており、フォーラムに記入された内容が自分の slack チャンネルに届くようになっている。

今回は firebase デプロイを使って、デプロイしています。
